Diagnostics

Overview of Available Diagnostics

AQUA provides a collection of built-in diagnostics to analyze climate model outputs.

  • Dummy
  • SSH variability
  • Teleconnections diagnostic
  • Tropical Cyclones detection, tracking and zoom in diagnostic
  • Tropical rainfall diagnostic
  • Global mean time series
  • Sea Ice Diagnostics
  • Radiation Budget Diagnostic
  • Atmospheric Global Mean Diagnostic
  • Ocean Circulation
  • Global Ocean

Creating Custom Diagnostics

AQUA allows users to create custom diagnostics to suit their specific needs. Custom diagnostics can be implemented as Python functions or classes and integrated into AQUA’s diagnostic framework.
